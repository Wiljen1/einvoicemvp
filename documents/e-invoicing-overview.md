# E-Invoicing MVP Approved Notes

The E-Invoice MVP is intended to help users answer questions about electronic invoicing workflows using only approved SharePoint folder documents.

The MVP should keep answers short, use business-friendly language, include source references, and clearly state when information is missing from the approved document set.

The application should show Codex availability, SharePoint or mock-folder access, the active approved folder, editable guardrails, and a chat interface.

For the local MVP, if SharePoint credentials are not configured, the approved mock folder is the local `documents` folder in this project.
